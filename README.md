## O projekcie

Implementacja algorytmu genetycznego (GA) rozwiązującego problem plecakowy 0/1, z benchmarkiem brute-force i badaniami eksperymentalnymi wpływu parametrów na jakość wyników.

Instancja: **18 przedmiotów**, W = 45, optimum = **84**.
